Análisis del dataset

Localización de las bases

Este fichero muestra las bases para encontrar las bicicletas públicas de Madrid.
  • Nombre del fichero: bases_bicimad.csv
  • CSV obtenido de: Portal de datos abiertos del Ayuntamiento de Madrid. BiciMAD. Bases del servicio público de bicicleta eléctrica de la ciudad de Madrid.
Nombre del dato Tipo Descripción Rango Problemas
Número String Número identificativo de la base - Existen una serie de números, y excepcionalmente,
acompañados de una letra
Gis_X Float Sin descripción [3.4E-38, 3.4E+38] El nombre del dato no es identificativo
Gis_Y Float Sin descripción [3.4E-38, 3.4E+38] El nombre del dato no es identificativo
Fecha de Alta Integer Fecha de alta de la base [-32768, 32767] Debería ser de tipo Date
Distrito String Indica el distrito mediante su código y el nombre del mismo - No se encuentran problemas
Barrio Integer Indica el barrio mediante el código del distrito, su código y el nombre del mismo [-32768, 32767] No se encuentran problemas
Calle String Indica la localización de la base mediante el nombre de la calle,
seguido del tipo de la vía y de un nexo		 - No se encuentran problemas
Nº Finca String Indica el número en la calle donde se posiciona la base - Puede ser un número o S/N
Tipo de Reserva String Indica el tipo de reserva - No deja muy claro si existen más tipos de reserva, ya que
sólo aparece un único valor: BiciMAD
Número de Plazas Integer Indica la capacidad que tiene la base [-32768, 32767] No se encuentran problemas
LONGITUD Integer Indica la longitud donde se encuentra la base [-32768, 32767] No se encuentran problemas
LATITUD Integer Indica la latitud donde se encuentra la base [-32768, 32767] No se encuentran problemas
DIRECCION String Indica la localización de la base mediante el nombre de la calle,
seguido del tipo de la vía, el un nexo y el número de finca		 - No se encuentran problemas

Análisis de la licencia del dataset

Las condiciones generales permiten la reutilización de los documentos para fines comerciales y no comerciales. Se entiende por reutilización el uso de documentos que obran en poder del Ayuntamiento de Madrid, siempre que dicho uso no constituya una actividad administrativa pública.

Lo que esto implica, es que se pueden utilizar estos documentos para generar dinero (fines comerciales) o no.
Si se reutiliza esta información, existen unas series de pautas a seguir:

  • Está prohibido desnaturalizar el sentido de la información.
  • Hay que citar la fuente de los documentos.
  • Se debe indicar la fecha de la última actualización de los mismos.
  • No se podrá indicar, insinuar o sugerir que el Ayuntamiento de Madrid participa, patrocina o apoya la reutilización que se lleve a cabo con la información.
  • Los metadatos no deben ni ser modificados, ni ser eliminados.
  • En caso de información anonimizada por protección de datos personales u otros motivos, está expresamente prohibido realizar labores de re-identificación de personas a partir de estos datos y otras fuentes de datos e información posibles, pasadas, actuales o futuras.
