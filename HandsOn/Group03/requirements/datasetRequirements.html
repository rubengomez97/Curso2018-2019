Data Set Requirements

R1: Data are in the smart cities Domain, because the name clarifies it.
R2: Data are available as a CSV File, because it is clearly indicated in the webpage.
R3: Data have an open license so it can be published, because datos.madrid.es has an open licence.
R4: Data can be easily linked with generic real-world-entities, because it has geolocation also.
R5: Documentation exits for the data, because the data is documented with a respective PDF.
